# User API
> A simple user api to manage user authentication


## Environment Variables
Create `.env` file in project directory and fill all variables values with yours.
```
NODE_ENV=
PORT=
MONGODB_URL=
TOKEN_SECRET_KEY=
```
## Install Dependencies
In the project directory, run:
```
npm i
```
## Run The Project
In the project directory, run:
```
npm start
```

## Features
- [x] User Sign Up
- [x] User Sign In
- [x] Reset Password
- [x] Verify Token
